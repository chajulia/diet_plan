A스크램블드에그 2개 + 방울토마토 + 아메리카노

C그릭요거트 150g (무가당) + 견과류 + 블루베리
D달걀프라이 2개 + 아보카도 1/2 + 통곡물빵 1장 (주 3회 이하)

E달걀국 + 잡곡밥 1/3공기 + 김치
F두부 부침 + 된장국 + 잡곡밥 1/3공기
G삶은 달걀 2개 + 미역국 + 잡곡밥 1/3공기
